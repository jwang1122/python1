  1. Add Extension (Markdown PDF, PDF Preview)

  3. Open Terminal
   click terminal menu > click "New Terminal" (ctrl+shift+`)
  4. Add Terminal
   go to the bottom of the terminal > click plus sign at top right corner
  5. Create a Python file
  6. Run Python file
